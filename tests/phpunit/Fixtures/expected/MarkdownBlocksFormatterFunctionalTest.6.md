### `VAR1`

Description without a leading space that goes on<br/>multiple lines.

And has a comment with no content.

Default value: `val1`

### `VAR2`

List header.
- "item1" - list item1.
- "item2" - list item2.<br/>Second line of list item2.
- "item3" - list item3.

Multiple lines after a list item.<br/>Second line.

And a third line.

Default value: `val2`
